# 导出导入镜像

---

### 导出镜像

```bash
docker save imageId > .

docker save -o image-file.tar imageName
```

### 导入镜像

```bash
docker load < image-file.tar

docker load -i ./image-file.tar
```

### 为镜像打上 `tag`
```bash
docker tag image-id name:version
```

### 运行镜像
```bash
docker run -itd -p 宿主port:容器port name
```
