## 如何使用css实现硬件加速？
> 硬件加速是指通过创建独⽴的复合图层，让GPU来渲染这个图层，从⽽提⾼性能

- ⼀般触发硬件加速的属性 CSS 有 `transform`, `opacity`, `filter`,为了避免2D动画在开始和结束的时候的操作，⼀般使repaint⽤`tranform:translateZ(0)`
