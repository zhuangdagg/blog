自定义容器 ​

信息

md
    ::: info
        // 信息内容
    :::

提示

md
    ::: tip
        // 提示内容
    :::

警告

md
    ::: warning
        // 信息内容
    :::

DANGER

md
    ::: danger
        // danger内容
    :::
详细信息
md
    ::: details
        // 详细信息内容
    :::

NOTE

强调用户在快速浏览文档时也不应忽略的重要信息。

TIP

有助于用户更顺利达成目标的建议性信息。

IMPORTANT

对用户达成目标至关重要的信息。

WARNING

因为可能存在风险，所以需要用户立即关注的关键内容。

CAUTION

行为可能带来的负面影响。
